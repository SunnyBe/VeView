# Third-Party Dependencies

The VeView SDK for Android uses the following third-party libraries:

## Main Dependencies

*   **AndroidX Core KTX**: [androidx.core:core-ktx](https://developer.android.com/kotlin/ktx)
*   **AndroidX Appcompat**: [androidx.appcompat:appcompat](https://developer.android.com/jetpack/androidx/releases/appcompat)
*   **AndroidX Datastore**: [androidx.datastore:datastore-preferences](https://developer.android.com/jetpack/androidx/releases/datastore)
*   **Google Material Components**: [com.google.android.material:material](https://github.com/material-components/material-components-android)
*   **Timber**: [com.jakewharton.timber:timber:5.0.1](https://github.com/JakeWharton/timber)
*   **OkHttp**: [com.squareup.okhttp3:okhttp:4.12.0](https://square.github.io/okhttp/)

## Test Dependencies

*   **Mockk**: [io.mockk:mockk:1.14.7](https://mockk.io/)
*   **Turbine**: [app.cash.turbine:turbine:1.2.1](https://github.com/cashapp/turbine)
*   **Google Truth**: [com.google.truth:truth:1.1.5](https://truth.dev/)
*   **AndroidX Test JUnit**: [androidx.test.ext:junit:1.1.5](https://developer.android.com/jetpack/androidx/releases/test)
*   **AndroidX Test Espresso Core**: [androidx.test.espresso:espresso-core:3.5.1](https://developer.android.com/training/testing/espresso)
*   **AndroidX Test Runner**: [androidx.test:runner:1.5.2](https://developer.android.com/jetpack/androidx/releases/test)
